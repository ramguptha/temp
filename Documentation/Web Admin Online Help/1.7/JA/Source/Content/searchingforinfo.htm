情報を検索する

の多くのページで、特定の検索基準を使用して情報を素早く見つけることができる検索フィールドがあります。

結果グリッド内の情報を検索するには:

検索を行うエリアに移動します。デフォルトで、すべての利用可能な情報が結果グリッドに含まれています。

特定の結果を検索するには、検索フィールドの隣にあるをクリックし、リストから検索基準を選択します。

利用可能な検索基準のリストは、操作中のエリア内の利用可能な列によって変わります。通常、結果グリッドで表示されている列は、検索基準として使用できます。

検索フィールドに、情報を探すために使用する検索キーワードのすべて、または一部を入力します。

検索結果は、入力内容に合わせて結果グリッドで動的に更新されます。
