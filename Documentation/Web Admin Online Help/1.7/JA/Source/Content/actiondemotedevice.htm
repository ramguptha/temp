アクション:非管理中デバイスへの降格

アクションを指定したスマートポリシーにモバイルデバイスが参加したときの、モバイルデバイスから MDM 制御を (該当する場合は Absolute Manage クライアントソフトウェアも) 削除するアクションを作成することができます。このアクションをモバイルデバイスに適用すると、Absolute Manage を通じた管理が行えなくなります。このアクションは、iOS および Android デバイスのみで利用できます。

Absolute Manage ではこのアクションを元に戻すことができません。管理外としたモバイルデバイスを再度 Absolute Manage で管理する場合、そのデバイスを再登録する必要があります。.

アクションを作成したら、割り当て可能なアイテム > アクション エリアの中で (お客様が展開している他のアクションと一緒に) それらを表示および管理することができます。

非管理中デバイスへの降格アクションを編集または複製する場合、以下のタスクの表を参照し、アクションで利用可能なフィールドの説明をご確認ください。

非管理中デバイスへの降格アクションを作成するには:

割り当て可能なアイテム > アクションエリアに移動します。（画面の左上にあるアイコンをクリックし、次にアクションをクリックすることもできます）

作業エリアのツールバーで、アクションを追加 > 非管理中デバイスへの降格をクリックします。

次の情報を入力します。入力必須フィールドにはアスタリスク (*) が付いています。

フィールド 説明

アクション名*

アクションの名前

説明

アクションの目的を明らかにする説明

この文章はに表示され、自分自身と他のアドミニストレーターの参考用です。

ターゲットプラットフォーム

このアクションが適用されるモバイルデバイスのターゲットプラットフォーム

次のいずれかを実行してください。

アクションをすぐにスマートポリシーに割り当てない場合、保存をクリックしてアクションを保存します。

アクションを保存して1つ以上のスマートポリシーに割り当てるには、保存してポリシーに割り当て をクリックします。
