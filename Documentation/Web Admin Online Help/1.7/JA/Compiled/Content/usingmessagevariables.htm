アクションで変数を使用する

モバイルデバイス用のメッセージまたは名前を含むアクションを作成する場合、特定のテキストフィールド内に、デバイスおよびユーザーに関連する変数を埋め込むことができます。これらの変数は、アクションが実行されるときに、対応する情報アイテムの内容に置換されます。

アクションで変数を使用するには、波括弧とドルマークの接頭辞で囲みます (例: ${MD_DeviceName})。

以下の表は、アクションに記載できる変数のリストです。

変数 対応する情報アイテムを含む

デバイス関連の変数

MD_CurrentCarrierNetwork

モバイルデバイスの現在のキャリアネットワーク

MD_DeviceModel

モバイルデバイスのモデル

MD_DeviceName

モバイルデバイス名

MD_HomeCarrierNetwork

モバイルデバイスのホームネットワーク

MD_IMEI国際移動体装置識別番号。すべての 3GPP (GSM、UMTS、および LTE) モバイルデバイスを一意に識別する15桁の番号。

モバイルデバイスの IMEI

MD_UDIDデバイスの一意の識別子。IPhone、iPod touch、iPad を一意に識別する 40 桁のひと続きの文字と数値。

モバイルデバイスの識別子 (UDID)

MD_PhoneNumber

モバイルデバイスの電話番号

MD_SerialNumber

モバイルデバイスのシリアル番号

MD_LastMDMHeartbeat

モバイルデバイスの最後の連絡

MD_IPAddress

モバイルデバイスのパブリック IP アドレス

MD_CellIPAddress

モバイルデバイスのセル IP アドレス

MD_WifiIPAddress

モバイルデバイスの Wi-Fi電子デバイスがインターネットに接続したり、電波を介して特定のエリア内の別の電子デバイスと無線で相互に通信することを可能にする技術。 IP アドレス

MD_WifiMACAddress

モバイルデバイスの Wi-Fi MAC アドレスメディアアクセス制御アドレス。12桁の16進数の番号であり、ハードウェアデバイスがネットワーク上で自分自身を一意に識別するために使用します。

MD_WifiNetwork

モバイルデバイスの Wi-Fi ネットワーク

MD_Model

モバイルデバイスのモデル番号

MD_OSVersion

モバイルデバイスの OSオペレーティング・システム。モバイルデバイス上のソフトウェアで、ハードウェアリソースを管理して各種アプリケーションに共通のサービスを提供します。 バージョン

MD_IsJailbroken

Jailbreak されたモバイルデバイス

MD_PasscodePresent

モバイルデバイスのパスコードが存在

MD_IsRoaming

モバイルデバイスがローミング中

MD_DataRoamingEnabled

モバイルデバイスのデータローミングが有効

MD_VoiceRoamingEnabled

モバイルデバイスボイスローミングが有効

ユーザー関連の変数

MDU_FirstName

デバイスユーザーのファースト・ネーム（名）

MDU_LastName

デバイスユーザーのラスト・ネーム（姓）

MDU_LogOnName

デバイスユーザーのログイン名

MDU_DisplayName

デバイスユーザーの表示名

MDU_EnrollmentUsername

デバイスユーザーの登録ユーザー名

MDU_EnrollmentDomain

デバイスユーザーの登録ドメイン

MD_PhoneNumber

デバイスユーザーの電話番号

MDU_EMail

デバイスユーザーの E メール

MDU_OrganizationalUnitPath

デバイスユーザーの組織単位のパス

MDU_OrganizationalUnitPath

デバイスユーザーの組織単位

MDU_IsMemberOf

デバイスユーザーは次のメンバーである

MDU_Department

デバイスユーザーの部署

MDU_Office

デバイスユーザーの事務所

MDU_Company

デバイスユーザーの会社

MDU_Street

デバイスユーザーの番地

MDU_City

デバイスユーザーの市区町村

MDU_State

デバイスユーザーの都道府県

MDU_Country

デバイスユーザーの国

MDU_ZIPCode

デバイスユーザーの郵便番号

また、カスタムフィールドを作成し、変数名をフィールドに割り当てることでカスタム変数を使用することもできます。
