Duplicating Actions

You can make a copy of an existing action and edit it to create another action.

To duplicate an action:

Navigate to the Assignable Items > Actions area. (You can also click in the top left corner of the screen and then click Actions.)

In the results grid, select the checkbox of the action you want to duplicate.

Click .

Edit the action's properties as required. For more information on the properties of a specific action type, see Working with Actions.

Do one of the following:

Click Save to save the action without immediately assigning it to a smart policy.

Click Save and Assign to Policies to save and assign the action to one or more smart policies.
