Duplicating Custom Fields

You can make a copy of an existing custom field and edit it to create another field.

To duplicate a custom field:

Click in the top left corner of the screen and then click Mobile Custom Fields.

In the results grid, select the checkbox of the custom field you want to duplicate.

Click .

Edit the field's properties as required. For more information on the properties of a custom field, see Creating Custom Fields.

Click Save.

Click OK to close the confirmation message.
