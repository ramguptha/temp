Gathering Computer Information

You can gather a wide range of information on your administered computers to help you manage your network.

Information about administered computers is regularly collected by the Absolute Manage agent, which sends the information to the Absolute Manage Server. This information is then retrieved by for display within the application. The process of collecting information is mostly automatic; however, certain types of information – such as font, printer, and startup item information – are collected on request only to conserve local processing power and network bandwidth. You can send a command to retrieve information on these specific items, as well as to manually update other computer information when you do not want to wait for the next scheduled update.

To manually gather computer information:

Navigate to a page in the Computers area that contains the computers you want to gather information about. For example, you can navigate to Computers > All Computers, then search for specific computers.

In the results grid, select the checkbox of each applicable computer and click Commands > Gather Inventory.

You can also issue the command for a single computer from its Computer Details page:

In the results grid, click the computer you want to gather information about. The computer's Computer Details page opens.

From the Commands sidebar, click Gather Inventory.

Computer information that has changed since the last update is always gathered when you perform this command. You can also select options for gathering additional information:

Force full inventory – Prompts agents on the selected computers to transmit all computer information (that is, not only information that has changed) to the server

Include font information – Prompts agents on the selected computers to transmit information on installed fonts

Include printer information – Prompts agents on the selected computers to transmit information on connected printers

Include startup item information (ОS X only) – Prompts agents on selected ОS X computers to transmit information on installed startup items

Include service information (Windows only) – Prompts agents on selected Windows computers to transmit information on running services

Click Gather Inventory.

Click OK to close the confirmation message.
