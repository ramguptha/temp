Action: Send VPP Invitation

You can create an action that sends an invitation to users to register in Apple's Volume Purchase Program (VPP) using their personal Apple ID. The invitation is sent to users when their mobile devices join smart policies you assign the action to. This action is available for iOS devices only.

Before sending this invitation to users, ensure they are already registered with your VPP account (for example, by using the Register User in VPP action).

After you create actions, you can view and manage them (along with all other actions that exist in your deployment) within the Assignable Items > Actions area.

If you are editing or duplicating a Send VPP Invitation action, refer to the table in the following task for a description of the fields available for the action.

To create a Send VPP Invitation action:

Navigate to the Assignable Items > Actions area. (You can also click in the top left corner of the screen and then click Actions.)

In the work area toolbar, click Add Action > Send VPP Invitation.

Enter the following information. Required fields are marked with an asterisk (*).

Field Description

Action name*

A name for the action

Description

A description explaining the purpose of the action

This text is shown within and is intended for your own reference and that of other administrators.

Target platforms

The target platforms of the mobile devices to which this action should apply

This action is available for iOS devices only.

VPP account

The VPP account in which you want users to register

This lists contains all VPP accounts that have been defined in the settings of your Absolute Manage Server.

Send invitation

The VPP invitation options. You can choose one or more of the following notification methods:

MDM dialog – Sends an invitation through an MDM dialog displayed on the mobile device. You can use this method with mobile devices running iOS 7.0.3 and higher only.

Web Clip – Sends an invitation using a Web Clip that is placed on the device's home screen

SMS – Sends an invitation through an SMS message. Enter the message in the field provided, up to a maximum of 140 characters. You can also use device and user-related variables in your message.

Email – Sends an invitation through email. Enter the subject and body of the email in the fields provided. You can also use device and user-related variables in the email body.

AbsoluteApps message – Sends an invitation through an AbsoluteApps message. Enter the body of the message in the field provided. You can also use device and user-related variables in your message. You can use this method only with mobile devices on which AbsoluteApps is installed.

In addition to device and user-related variables, you can also use the "MD_VPPInviteURL" variable within the invitation text, which displays a link to the Apple web page where users can enter their Apple ID to register in the VPP.

Do one of the following:

Click Save to save the action without immediately assigning it to a smart policy.

Click Save and Assign to Policies to save and assign the action to one or more smart policies.
