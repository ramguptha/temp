Viewing Action Details

You can view the details of actions that have been created within , including the properties and policies that are associated with each action.

To view the details of an action:

Navigate to the Assignable Items > Actions area. (You can also click in the top left corner of the screen and then click Actions.)

In the results grid, click the action whose details you want to view.

The Action Details page opens. The following table provides a summary of the information available on this page:

Field Description

All actions

Name

The name under which the action is stored in Absolute Manage

Description

Descriptive text explaining the purpose of the action. This text is shown within and is intended for your own reference and that of other administrators.

Type

The type of action being performed

Target platforms

The mobile device platforms to which the action applies

Last modified

The date and time when the action was last modified

Send Message To Device

Message text

The text that is sent to mobile devices that trigger the action. This message appears onscreen on the mobile device.

Send E-Mail

To

The email address to which the e-mail should be sent

CC

The email addresses to which the e-mail should be copied, if any

Subject

The subject of the email

Message text

The body of the email

Send SMS

Message text

The message to be sent, up to 140 characters in length

Phone number

The telephone numbers to which the SMS text message should be sent

Set Roaming Options

Voice roaming

The voice roaming option to set on mobile devices entering a smart policy

Data roaming

The data roaming option to set on mobile devices entering a smart policy

Set Activation Lock Options

Activation lock

Indicates whether the activation lock can be enabled on iOS devices through the “Find My iPhone” or “Find My iPad” setting

Set Wallpaper

Wallpaper options

Indicates which screens on the mobile device the wallpaper should apply to

Image

A preview of the uploaded wallpaper image

Image dimensions

The dimensions of the uploaded wallpaper image

Set Device Name

Device name

The new device name the action applies to mobile devices

Set Custom Field Value

Custom field

The custom field for which the value is being set

Value

The value applied to the custom field on mobile devices on which the action is performed

Set Attention Mode

Attention mode

Indicates whether the action enables or disables attention mode

Text

The message shown on the screen of mobile devices while attention mode is enabled

Register User In VPP and Send VPP Invitation

Subject

The subject text for the invitation email

Message text

The text of the invitation message sent using email or AbsoluteApps

SMS text

The text of the invitation message sent through SMS

Send invitation

The channels over which the action sends the VPP registration notice to users

VPP account

The VPP account under which the action registers mobile device users

Retire User From VPP

VPP account

The VPP account from which the action removes mobile device users

Remove Configuration Profile

Profile

The configuration profile that the action removes from mobile devices

You can also perform the following tasks from this page:

Duplicate the action

Edit the action

Delete the action

Click the Policies tab to view the smart policies that the action is assigned to. You can also perform the following tasks from this tab:

Add the action to smart policies from this tab by clicking the Add Action To Policies button.

Remove the action from smart policies by selecting the checkbox of the applicable smart policies and clicking .

Edit the policy assignment rules for the action by selecting the checkbox of the applicable smart policy and clicking .

You can quickly find specific results on any of these pages by using the search field at the top of the page to narrow the items listed by specific criteria (for example, by policy name). You can also customize the column layout of these pages to show the information that is most relevant to you.
