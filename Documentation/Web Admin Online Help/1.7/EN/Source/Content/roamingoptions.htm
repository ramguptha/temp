Setting Roaming Options for Devices

You can issue a command to activate or deactivate voice and data roaming on mobile devices. This command is available only for iOS devices running iOS 5 or higher.

After you set roaming options for a device, the device's user is not prevented from changing these settings.

After you issue the command, a device's roaming settings are changed as soon as the device next contacts the notification server of its OS vendor. If the device is connected to a Wi-Fi or mobile network, this action is usually carried out within a minute; otherwise, it happens when the device reconnects to a network.

To set roaming options for mobile devices:

Navigate to a page in the Mobile Devices area that contains the devices whose roaming options you want to change. For example, you can navigate to Mobile Devices > All iOS Devices, then search for specific devices.

In the results grid, select the checkbox of each applicable device and click Commands > Set Roaming Options.

You can also issue the command for a single iOS device from its Device Details page:

In the results grid, click the device whose roaming options you want to change. The device's Device Details page opens.

From the Commands sidebar, click Set Roaming Options.

Set the roaming options for the devices by selecting or clearing the Enable voice roaming and Enable data roaming checkboxes.

Click Set Roaming Options.

Click OK to close the confirmation message.
