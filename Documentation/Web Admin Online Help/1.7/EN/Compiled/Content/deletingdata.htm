Erasing Devices

You can remotely erase all user data on a mobile device as well as all applications the user has installed, effectively resetting the device to its factory condition.

WARNING!   This actionAllows you to specify what Absolute Manage should do when a mobile device enters a smart policy. is not reversible. You cannot recover the erased information from the mobile device (although recovering the data from a backup system, if one exists within your organization, may be possible). Erasing a device without the consent of the user may expose you to legal liability.

After you issue the command, a device is erased as soon as it next contacts the notification server of its OSOperating System. The software on mobile devices that manages hardware resources and provides common services for applications. vendor. If the device is connected to a Wi-FiA technology that allows electronic devices to connect to the Internet or communicate with one another wirelessly within a particular area over radio waves. or mobile network, this action is usually carried out within a minute; otherwise, it happens when the device reconnects to a network.

After a device is erased, you can no longer administer that device through Absolute Manage until you enroll it again or it has been completely restored from a backup containing the Absolute Manage enrollment profile.

To erase mobile devices:

1. Navigate to a page in the Mobile Devices area that contains the devices you want to erase. For example, you can navigate to Mobile Devices > All Mobile Devices, then search for specific devices.
2. In the results grid, select the checkbox of each applicable device and click Commands > Erase Device.

You can also issue the command for a single device from its Device Details page:

a) In the results grid, click the device you want to erase. The device's Device Details page opens.
b) From the Commands sidebar, click Erase Device.
3. If any of the selected devices have internal SDSecure Digital. A memory card format used in portable devices such as smartphones. cards, you are prompted to select one of the following options:
● Erase internal storage only
● Erase internal storage and SD card
4. Click Erase Device.

IMPORTANT   Clicking Erase Device issues the command to erase data on the device, which you cannot reverse.

5. Click OK to close the confirmation message.
