Viewing a List of Registered AM Servers

The AM Servers grid on the AM Servers tab shows details about the AM Servers currently registered in your installation of Web Admin. The grid includes the following details: 

● AM Server Name: The name of the AM Server, as registered with Web Admin.
● Port Number: The port number that the AM Servers use for Self-Service Portal user logins.
● SSP Default: Information on whether the AM Server is the default server to which SSP users connect.
