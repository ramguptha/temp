Managing AM Servers in Your Web Admin Installation

You can use Web Admin Configuration Tool to view a list of existing AM Servers registered in your Web Admin installation, register new AM Servers, and set a default AM Server for your Self-Service Portal user logins.

You can use the AM Servers tab in the Web Admin Configuration Tool to perform the following tasks: 

● Viewing a List of AM Servers
● Registering a New AM Server
